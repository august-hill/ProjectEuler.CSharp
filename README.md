# Project Euler Solutions - C#

100 [Project Euler](https://projecteuler.net/) solutions in C#, part of a cross-language benchmark suite with standardized benchmarking across 11 languages.

See [ProjectEuler.Benchmarks](https://github.com/august-hill/ProjectEuler.Benchmarks) for cross-language performance comparisons.

## Prerequisites

- **Runtime**: .NET 10+
- **Platform**: macOS / Linux / Windows

## Running

```bash
# Run a single solution
cd problem_001
dotnet run -c Release

# Run all benchmarks (from repo root)
./run_all.sh
```

## Project Structure

```
ProjectEuler.CSharp/
  problem_001/
    Program.cs              # Solution + benchmark for problem 1
    Bench.cs                # Shared benchmarking class
    problem_001.csproj
  problem_002/
    Program.cs
    Bench.cs
    problem_002.csproj
  ...
  problem_100/
```

Each `Program.cs` uses `Bench.cs` and outputs timing results in a standardized format consumed by the Benchmarks aggregator.
